THE DESIGN PROCESS WITH ALL PROJECTS/CASE STUDIES

Discover Phase

This phase included research development and execution with users and stakeholders, heuristic evaluations, proto persona and user persona development, affinity diagraming and empathy mapping to understand our users and their pain points.

Define Phase

This phase included analysis of research findings and defining the problem statement, creating a journey map, storyboarding, identifying key issues we could adress through the "I Wish, I Like, What If" model and a feature prioritization diagram and creating a user flow

Development Phase

This phase included competator analysis's, user flow iterations, doing hand sketches, creating low fidelity wireframes, designing digital wireframe prototypes, testing via user tests and made iterations to prototype from test results.

Refine and Deliver Phase

This phase included high fidelity prototyping, Guerrilla testing, iterating, case study development and presenting the case study to the class.
